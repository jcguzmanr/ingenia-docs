## Estructura ClickUp — 5 Spaces Unificados

```
INGENIA LATAM (Workspace)
│
├─ ESTRATEGIA (Pensar → Decidir)
│  ├─ Comités Estratégicos
│  ├─ Diagnosis & Research
│  └─ Consulting Brain (hipótesis, frameworks, aprendizajes)
│
├─ CONSULTIVA PIPELINE (Flujo de Valor)
│  ├─ Pipeline de Empresas (46 + nuevos, 10 etapas + AI Fields)
│  ├─ Directorio de Contactos (94 + nuevos)
│  ├─ Licitaciones (formal processes)
│  └─ SOWs & Propuestas (en desarrollo/negociación/ganadas)
│
├─ CLIENTES (Operación & Delivery)
│  ├─ [Cliente A] → Fase 1, 2, 3, 4
│  ├─ [Cliente B] → Fase 1, 2, 3, 4
│  └─ ... (idem para cada cliente)
│
├─ OPERACIÓN (Ejecutar → Aprender)
│  ├─ Tareas Operacionales (trabajo diario)
│  ├─ Reuniones & Rituales (comités, status, eventos)
│  └─ KPIs & Reportería (métricas consultivas)
│
└─ CONOCIMIENTO (Aprender → Mejorar)
   ├─ Research Acumulado (industrias, verticales, mercados)
   ├─ Casos y References (learnings, best practices)
   └─ Plantillas & Procesos (reutilizables)
```
