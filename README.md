# Ingenia Docs

Documentos y páginas internas Ingenia Latam (hub, pre-reunión, presentaciones).

## Enlace único (GitHub Pages)

Cuando actives **GitHub Pages** en este repo:

- **URL del sitio:** https://jcguzmanr.github.io/ingenia-docs/
- **Hub:** https://jcguzmanr.github.io/ingenia-docs/
- **Pre-reunión interactiva:** https://jcguzmanr.github.io/ingenia-docs/prereunion-18marzo-interactivo.html

### Cómo activar GitHub Pages

1. En el repo: **Settings** → **Pages**
2. En **Source** elige **Deploy from a branch**
3. **Branch:** `main` — **Folder:** `/ (root)` — **Save**
4. En unos minutos el sitio estará en la URL de arriba.

Desde ahí puedes ir sumando más páginas (nuevos HTML) y enlazarlas desde el `index.html` o entre sí.
